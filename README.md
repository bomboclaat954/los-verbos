# Los Verbos

Los Verbos is a simple Python program created to help Spanish learners with verb conjugation. It works with regular verbs but there's also a JSON file containing irregular ones. You can also add your own irregular verbs.

# Usage

Clone the repo

```bash
$ git clone https://github.com/bomboclaat954/los-verbos
```

Go to cloned folder

```bash
$ cd los-verbos
```

Now run it in your terminal

```bash
$ python3 main.py
```

# Adding own verbs

To add your own irregular verbs run the program, and instead of typing a verb type `\a`.

# Notes

1. The app's language is Spanish but if you speak English, you'll understand everything.
